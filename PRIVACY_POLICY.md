## 6. Security

### 6.1 Security Measures We Implement:

- **Encryption:** HTTPS/TLS for all data transmissions
- **Authentication:** Secure login with password encryption
- **Authorization:** Role-based access controls
- **Data Protection:** Encrypted databases and backups
- **Access Control:** Limited staff access to personal data
- **Monitoring:** Continuous security monitoring and intrusion detection
- **Regular Audits:** Quarterly security assessments
- **Secure APIs:** Authorized endpoints with token validation

### 6.2 Your Security Responsibilities:

- Keep your password confidential
- Do not share account credentials
- Log out when using shared devices
- Report suspicious activity immediately
- Keep app updated to latest version
- Use device lock screen protection

## 8. Location Data

### 8.1 Real-Time Location:
- Collected only when you are online
- Used to match with riders
- Shared with rider during active trip
- Deleted after trip completion (90 days retention for history)

### 8.2 Location Permissions:
- App requests permission on first use
- Can be enabled/disabled in device settings
- Cannot use app features (matching) without location while online

### 8.3 Location Tracking:
- Ambient location collection for active trips only
- No background tracking when offline
- No geofencing or long-term location history
